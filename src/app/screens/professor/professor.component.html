Área do Professor

Descubra passo a passo como usar a ferramenta Zoom* para as suas aulas. Começamos por esta ferramenta porque foi a indicada como prioritária no inquérito que fizemos aos professores. Em breve publicaremos guias para outras plataformas, além de novos conteúdos que o ajudarão a usar a tecnologia para criar aulas à distância cada vez mais motivadoras para os alunos e para si.
